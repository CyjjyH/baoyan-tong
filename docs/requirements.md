# 产品需求文档 — 保研信息聚合平台

> 版本：v1.0 | 创建日期：2026-06-30 | 状态：待开发

---

## 1. 产品概述

### 1.1 产品名称
**待定**（建议：保研通 / 研招汇 / OfferCamp）

### 1.2 一句话描述
帮助大学生实时筛选各高校夏令营/预推免通知的信息聚合网站。

### 1.3 目标用户
- **主要用户**：大三准备保研/考研的大学生
- **次要用户**：大二提前了解保研信息的同学

### 1.4 核心价值
- 不用每天刷几十个高校官网，一个网站看全网通知
- AI 自动搜集 + 结构化展示，省时省力
- 筛选条件精准，只看到自己关心的通知

---

## 2. 功能需求

### 2.1 首页 — 搜索 & 筛选（P0 最高优先级）

| 功能点 | 需求描述 |
|--------|----------|
| 关键词搜索 | 输入框，支持搜索通知标题和内容 |
| 专业筛选 | 文本输入，模糊匹配专业名称 |
| 学校等级筛选 | 多选复选框：985 / 211 / 双一流 / 普通本科 |
| 学校名称筛选 | 文本输入或下拉选择具体学校 |
| 省份城市筛选 | 省/市两级联动选择 |
| 通知类型筛选 | 标签切换：全部 / 夏令营 / 预推免 |
| 报名时间筛选 | 日期范围选择器，按报名起止日期筛选 |
| 筛选结果列表 | 卡片式展示，每条展示关键信息摘要 |
| 结果排序 | 按发布时间 / 报名截止日期排序 |

### 2.2 通知详情页（P0）

| 展示字段 | 说明 |
|----------|------|
| 学校名称 + 学院 | 必填 |
| 专业/方向 | 必填 |
| 通知类型 | 夏令营 / 预推免 |
| 报名开始时间 | 必填 |
| 报名截止时间 | 必填 |
| 活动时间 | 选填 |
| 报名链接 | 外链 |
| 通知原文链接 | 外链 |
| 申请要求 | 文本（如 GPA 要求、英语要求等） |
| 所需材料 | 文本（如个人陈述、推荐信等） |
| 学校等级 | 985/211/双一流/普通 |
| 省份/城市 | 学校所在地 |

### 2.3 用户系统（P1）

| 功能 | 描述 |
|------|------|
| 注册 | 邮箱 + 密码注册，发送验证邮件 |
| 登录 | 邮箱 + 密码登录 |
| 找回密码 | 邮箱重置 |
| 收藏通知 | 详情页一键收藏，个人中心查看 |
| 取消收藏 | 列表或详情页取消 |
| 订阅提醒 | 设置筛选条件组合，匹配时邮件通知 |
| 个人中心 | 我的收藏、我的订阅、账号设置 |

### 2.4 后台管理（P1）

| 功能 | 描述 |
|------|------|
| 通知列表 | 查看/搜索/筛选所有通知 |
| 通知编辑 | 新增/修改/删除通知 |
| 投稿审核 | 审核用户投稿，通过/驳回 |
| AI结果抽查 | 查看 AI 自动入库的通知，快速确认或修正 |

### 2.5 AI 自动搜集（P2）

| 功能 | 描述 |
|------|------|
| 定时搜索 | 每天定时搜索各高校官网 |
| 信息提取 | AI 提取结构化信息（学校、专业、时间等） |
| 自动入库 | 提取结果自动写入待审核库 |
| 去重判断 | 同一通知不重复录入 |

---

## 3. 非功能需求

| 项目 | 要求 |
|------|------|
| 响应式设计 | 手机/平板/PC 均需适配 |
| 性能 | 首页加载时间 < 3 秒 |
| 安全 | 密码加密存储，防 SQL 注入/XSS |
| 可用性 | 筛选操作不超过 3 步就能找到目标通知 |
