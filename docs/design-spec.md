# 设计规范 — 保研信息聚合平台

> 版本：v1.0 | 创建日期：2026-06-30

---

## 1. 设计原则

- **简洁直观**：用户 3 步内找到目标通知
- **信息层次清晰**：重要信息突出，次要信息不干扰
- **蓝白配色**：专业、可信赖、清爽

## 2. 色彩系统

| 用途 | 色值 | Tailwind 类 |
|------|------|-------------|
| 主色（按钮、链接、强调） | `#2563EB` | `blue-600` |
| 主色深（悬停） | `#1D4ED8` | `blue-700` |
| 主色浅（背景） | `#EFF6FF` | `blue-50` |
| 背景白 | `#FFFFFF` | `white` |
| 页面背景 | `#F8FAFC` | `slate-50` |
| 卡片背景 | `#FFFFFF` | `white` |
| 标题文字 | `#0F172A` | `slate-900` |
| 正文文字 | `#334155` | `slate-700` |
| 次要文字 | `#94A3B8` | `slate-400` |
| 边框 | `#E2E8F0` | `slate-200` |
| 成功/进行中 | `#16A34A` | `green-600` |

## 3. 字体

| 用途 | 字体 | 字号 |
|------|------|------|
| 页面标题 | 系统默认字体 Bold | `text-2xl` (24px) |
| 卡片标题（学校名） | 系统默认字体 Semibold | `text-lg` (18px) |
| 正文 | 系统默认字体 Regular | `text-sm` (14px) |
| 辅助信息 | 系统默认字体 Regular | `text-xs` (12px) |

> 使用系统默认字体栈，无需加载 Web 字体，加载速度最快。

## 4. 间距系统

- 统一使用 Tailwind 默认间距：`p-4` (16px), `gap-3` (12px), `space-y-4` (16px)
- 卡片间距：12px / 16px
- 页面内容最大宽度：`max-w-6xl` (1152px)，居中

## 5. 圆角 & 阴影

- 卡片圆角：`rounded-lg` (8px)
- 按钮圆角：`rounded-md` (6px)
- 卡片阴影：`shadow-sm`（轻微阴影，hover 时 `shadow-md`）

## 6. 组件设计

### 6.1 导航栏（Header）
- 顶部固定，白色背景，底部细边框
- 左侧：Logo + 网站名称
- 右侧：登录/注册按钮（未登录）或 头像+用户名（已登录）
- 高度：64px

### 6.2 搜索区
- 页面顶部，浅蓝背景 `blue-50`
- 关键词搜索框 + 筛选条件（可折叠）
- 筛选标签水平排列，已选条件显示为可删除标签

### 6.3 通知卡片
- 白色卡片，左侧蓝色细线或顶部蓝色小条
- 第一行：学校名称 + 学院 + 通知类型标签
- 第二行：专业方向
- 第三行：报名时间 → 截止日期高亮
- 底部：学校等级标签 + 省份 + 收藏按钮
- Hover：轻微上浮 + 阴影加深

### 6.4 按钮
- 主按钮：蓝底白字 (`bg-blue-600 text-white`)
- 次按钮：白底蓝边框 (`border border-blue-600 text-blue-600`)
- 危险按钮：红底白字
- 尺寸：`h-10 px-4` (40px 高)

### 6.5 表单
- 输入框：白底，灰色边框，focus 时蓝色边框
- 标签：输入框上方，深灰色文字
- 错误提示：红色小字，输入框下方

## 7. 响应式断点

| 断点 | 宽度 | 适配 |
|------|------|------|
| 手机 | < 768px | 单列，筛选条件折叠为弹窗 |
| 平板 | 768-1024px | 两列，筛选在侧边 |
| PC | > 1024px | 三列或宽列表，筛选在左侧固定 |
