## 简化决策

用户选择"先做最小可用版本"路线：
- ❌ 不写管理后台页面
- ❌ 不写 REST API 接口
- ✅ 数据库 seed 数据直接跑起来
- ✅ Server Component 直连 DB，省掉 API 层
- ✅ 前台页面直接用 Prisma Studio 录入真实数据（npx prisma studio）

## 项目当前状态

```
npm run dev  →  http://localhost:3000  首页展示 6 条通知卡片
点击卡片     →  /notice/[id]           详情页展示完整信息
npx prisma studio → http://localhost:5550  可视化数据库管理
```
